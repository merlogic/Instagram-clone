# Instagram Clone 

This is a Flutter-based Instagram clone focused on clean design and smooth user experience. It uses a modern UI with glass-style elements and simple navigation across screens.

## Features

* Feed with interactive posts 
* Story section with gradient borders 
* Search screen with sample user profiles
* Glass-style bottom navigation bar
* Multiple sections: Home, Reels, Messages, Search, and Profile

##  Tech Stack

* Flutter
* Dart
* Material Icons

##  Project Structure

```text
lib/
├── features/
│   ├── feed/          # Feed and stories
│   ├── search/        # Search screen
│   └── profile/       # Profile layout
├── shared/
│   └── widgets/       # Reusable components
└── main_screen.dart   # Navigation logic
```
